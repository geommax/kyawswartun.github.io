# Kyaw Swar Htun - Robotics, AI, Embedded IoT Engineer Portfolio

A modern, responsive one-page portfolio website showcasing the expertise of Kyaw Swar Htun in Robotics, Artificial Intelligence, and Embedded IoT Engineering. This portfolio is designed for deployment on GitHub Pages.

## Features

- Clean, professional design with a tech-focused aesthetic
- Fully responsive layout that works on all devices
- Smooth scrolling navigation
- Animated elements for enhanced user experience
- Organized sections for:
  - Professional summary
  - Technical skills (Robotics, AI, IT Systems)
  - Project showcase
  - Work experience timeline
  - Contact information

## Deployment to GitHub Pages

1. Create a new repository on GitHub (or use an existing one)
2. Push the contents of this folder to your repository
3. Go to your repository settings
4. Scroll down to the "Pages" section
5. Under "Source", select "Deploy from a branch"
6. Select "main" branch and "/ (root)" folder
7. Click "Save"
8. Your site will be published at `https://[username].github.io/[repository-name]/`

## Customization

To further customize this portfolio:

1. Edit `index.html` to update personal information, skills, projects, and experience
2. Modify `styles.css` to change colors, fonts, and layout
3. Update `script.js` for any interactive functionality
4. Replace placeholder icons and images with your own
